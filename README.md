# HR Management System

A Windows Forms application built with C# and .NET for managing employees.

## Features
- Add, Update, Delete employees
- Filter employees by any column
- Sort by any field (Asc/Desc)
- Double-click on row to load employee data
- Combo box filtering by Employee ID

## Tech Stack
- C# / .NET
- Windows Forms
- SQL Server
- 3-Layer Architecture (DAL / BLL / UI)

## Database Tables
- Employees
- Departments
- Countries

## How To Run
1. Clone the repository
2. Update connection string in `Settings.cs`
3. Run the SQL script in 'HR_Project.pdb` folder
4. Build and run the project
